About Me

MY
JOURNEY

I'm Amit Rai, a BCA student passionate about Java development and software engineering. My journey started with curiosity — opening a code editor on my phone and writing my first "Hello, World!" in Java.

What makes my story different is that I've been learning mostly using my mobile phone. No expensive laptop, no dedicated setup — just consistency, YouTube, documentation, and a strong will to build.

"I believe real learning happens when you build real things — not when you just watch tutorials."

I focus on practical learning: every concept I study gets turned into a mini project. From a Student Grade Calculator to a Banking System — each project taught me something new about Java, OOP, and how software actually works.
